Процесс работы с задачами

Действия, которые можно совершать на форме с задачами:

1. Добавление задачи: нажать на кнопку "Добавить задачу", справа появится окно добавления задачи.
Выберите с помощью комбо бокса категорию создаваемой задачи.
В поле название введите название, в поле описание введите описание задачи, также введите дедлайн
и отметьте чекбоксом статус задачи.Если статус не выполенно, отмечать ничего не нужно. Нажмите кнопку "Сохранить"

2. Редактировние задачи: нажать на созданную задачу. Откроется окно редактирования задачи,
нужные данные можно поменять и нажать кнопку "Сохранить"

3. Удаление задачи: нажать на созданную задачу. Откроется окно редактирования задачи,
нажать на иконку муссорного ведра и подтвердить удаление задачи

4. Фильтрация задач: с помощью комбо бокса выбрать категорию

При нажатии на три точки в правом верхнерм углу, произойдёт переход в настройки

При нажатии на кнопку меню "Заметки", произойдёт переход на форму работы с заметками

При нажатии на кнопку меню "Аналитика", произойдёт переход на форму аналитики

При нажатии на кнопку меню "О программе", произойдёт переход на форму о программе
